  <name>Task 1: Create procedural insect shader</name>
  <files>shaders/insect_procedural.gdshader</files>
  <action>
Create `shaders/insect_procedural.gdshader` (spatial shader) that generates insect‑like patterns.

Key features:
1. Base color from uniform `base_color: vec4`
2. Pattern complexity from uniform `pattern_complexity: float`
3. Use multiple layers of `FastNoiseLite` (built‑in Godot noise functions) to create stripes, spots, gradients.
4. Mix patterns based on UV coordinates (or world position for primitive shapes).
5. Include parameters for scale (so patterns scale with insect size).
6. Use fragment shader to output final color.

Shader structure:
```
shader_type spatial;

uniform vec4 base_color: source_color = vec4(1.0);
uniform float pattern_complexity: hint_range(1, 10) = 5.0;
uniform float pattern_scale: hint_range(0.1, 10.0) = 1.0;

void fragment() {
    // Generate noise patterns
    float noise1 = noise(vec3(UV * pattern_complexity * pattern_scale, 0.0));
    float noise2 = noise(vec3(UV * pattern_complexity * pattern_scale * 2.0, 1.0));
    
    // Combine patterns
    float stripes = sin(UV.x * pattern_complexity * 10.0) * 0.5 + 0.5;
    float spots = step(0.7, noise1);
    
    // Mix with base color
    vec3 pattern_color = mix(base_color.rgb, vec3(0.2), spots * 0.3);
    pattern_color = mix(pattern_color, base_color.rgb * 1.2, stripes * 0.2);
    
    ALBEDO = pattern_color;
}
```

Make patterns insect‑like: stripes for cricket, spotted for beetle, gradient for spider. Use conditionals based on pattern type if needed.

Test shader in Godot's shader editor.
