<task type="auto">
  <name>Task 1: Create CameraFollow script</name>
  <files>res://scripts/camera_follow.gd</files>
  <read_first>
    - Godot documentation on Camera3D, Node3D, interpolation.
    - Existing camera node structure in the scene.
  </read_first>
  <action>
    1. Create script `camera_follow.gd` extending `Node3D` (or `Camera3D` if attached directly to camera). We'll assume it's a sibling node that controls camera position.
    2. Define exported variables:
       ```gdscript
       @export var target: Node3D
       @export var follow_speed: float = 5.0
       @export var distance_offset: Vector3 = Vector3(0, 2, -5)  # camera offset behind target
       ```
    3. In `_process(delta)`:
       - If target is not null:
         - Calculate desired position: `target.global_position + distance_offset`
         - Interpolate current position towards desired position using `lerp` or `global_position = global_position.lerp(desired_position, follow_speed * delta)`
         - Optionally make camera look at target.
       - If target is null, do nothing (camera stays).
    4. Save script.
    5. Attach CameraFollow script to the camera's parent node (e.g., "CameraPivot") or create a new node as child of camera.
  </action>
  <verify>

    - CameraFollow node exists in scene and script attached.
    - Script compiles without errors.
  </verify>
  <done>CameraFollow script ready to smoothly follow a target.</done>

<task type="auto">
  <name>Task 2: Create CameraManager script</name>
  <files>res://scripts/camera_manager.gd</files>
  <read_first>
    - res://scripts/signal_bus.gd (existing signals)
    - res://scripts/camera_follow.gd
  </read_first>
  <action>
    1. Create script `camera_manager.gd` extending `Node`.
    2. Define exported variables:
       ```gdscript
       @export var camera_follow: CameraFollow
       @export var tracking_enabled: bool = false
       ```
    3. Define function `set_tracking_mode(enabled: bool)`:
       - Set `tracking_enabled = enabled`
       - Emit `SignalBus.tracking_mode_toggled.emit(enabled)`
       - If not enabled, set `camera_follow.target = null`
    4. In `_ready`:
       - Connect `SignalBus.insect_selected` to a function `_on_insect_selected`.
       - Connect `SignalBus.insect_deselected` to a function `_on_insect_deselected`.
    5. Implement `_on_insect_selected(insect_node)`:
       - If `tracking_enabled`:
         - `camera_follow.target = insect_node`
    6. Implement `_on_insect_deselected`:
       - If `tracking_enabled`:
         - `camera_follow.target = null` (or keep target? decide to keep last insect)
    7. Save script.
    8. Add CameraManager node to scene (as child of root) and assign camera_follow reference.
  </action>
  <verify>
    <automated>test -f "res://scripts/camera_manager.gd" &amp;&amp; grep -q "tracking_enabled" "res://scripts/camera_manager.gd" &amp;&amp; echo "CameraManager script exists with tracking logic"</automated>

    - CameraManager node exists in scene with camera_follow assigned.
    - Script compiles and connects to SignalBus signals.
  </verify>
  <done>CameraManager manages tracking mode and assigns target to CameraFollow.</done>
