<task type="auto">
  <name>Task 2: Create HintManager</name>
  <files>res://scripts/hint_manager.gd</files>
  <read_first>
    - res://scripts/signal_bus.gd
    - Godot documentation on queues and timers.
  </read_first>
  <action>
    1. Create script `hint_manager.gd` extending `Node`.
    2. Define variables:
       ```gdscript
       var hint_queue: Array = []
       var current_hint: String = ""
       var is_showing: bool = false
       @export var hint_duration: float = 5.0
       ```
    3. Define function `add_hint(hint_text: String)`:
       - Append to `hint_queue`.
       - If not `is_showing`, call `show_next_hint()`.
    4. Define function `show_next_hint()`:
       - If `hint_queue.is_empty()`: return.
       - `current_hint = hint_queue.pop_front()`
       - `is_showing = true`
       - Call `HintPanel.display_hint(current_hint)` (assuming HintPanel is autoload or referenced).
       - Start a timer for `hint_duration`, then call `hide_hint()`.
    5. Define function `hide_hint()`:
       - Call `HintPanel.hide_hint()`.
       - `is_showing = false`
       - Call `show_next_hint()` if queue not empty.
    6. In `_ready`:
       - Connect `SignalBus.behavior_observed` to a function `_on_behavior_observed`.
    7. Implement `_on_behavior_observed(behavior_type, insect_node)`:
       - Map behavior_type to hint text (e.g., `"The " + insect_node.species + " is " + behavior_type + "."`).
       - Call `add_hint(hint_text)`.
    8. Save script.
    9. Add HintManager node to scene (as child of root) or make it autoload (optional).
  </action>
  <verify>
    <automated>test -f "res://scripts/hint_manager.gd" &amp;&amp; grep -q "add_hint" "res://scripts/hint_manager.gd" &amp;&amp; echo "HintManager script exists with queue management"</automated>
    - File `res://scripts/hint_manager.gd` exists with queue management.
    - Script connects to `behavior_observed` signal.
    - `add_hint` and `show_next_hint` functions implemented.
    - HintManager node exists in scene.
  </verify>
  <done>HintManager queues hints based on behavior events.</done>
</task>
