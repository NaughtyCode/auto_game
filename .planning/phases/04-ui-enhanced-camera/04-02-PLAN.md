<task type="auto">
  <name>Task 2: Create SelectionManager script</name>
  <files>res://scripts/selection_manager.gd</files>
  <read_first>
    - res://scripts/signal_bus.gd (signals)
    - Godot input handling and raycasting documentation.
  </read_first>
  <action>
    1. Create script `selection_manager.gd` extending `Node`.
    2. In `_ready`, set `process_mode` to `PROCESS_MODE_ALWAYS` to capture input even when paused.
    3. In `_unhandled_input(event)`, detect left mouse button press:
       - If event is `InputEventMouseButton` with `button_index == MOUSE_BUTTON_LEFT` and `pressed`.
       - Perform a raycast from camera through mouse position:
         ```gdscript
         var camera = get_viewport().get_camera_3d()
         var mouse_pos = get_viewport().get_mouse_position()
         var ray_length = 1000
         var from = camera.project_ray_origin(mouse_pos)
         var to = from + camera.project_ray_normal(mouse_pos) * ray_length
         var space_state = get_world_3d().direct_space_state
         var query = PhysicsRayQueryParameters3D.create(from, to)
         query.collision_mask = 1  # assuming insect collision layer is 1
         var result = space_state.intersect_ray(query)
         ```
       - If result has collider, check if collider's parent or node has "Insect" group or script:
         - Emit `SignalBus.insect_selected.emit(collider.get_parent())` (or collider).
       - If no collider, emit `SignalBus.insect_deselected.emit()`.
    4. Add a variable `current_selected` to store reference (optional).
    5. Save script.
    6. Add SelectionManager node to main scene (e.g., as child of root). Provide instruction: "Add a Node as child of root and attach SelectionManager script."
  </action>
  <verify>
    <automated>test -f "res://scripts/selection_manager.gd" &amp;&amp; grep -q "_unhandled_input" "res://scripts/signal_bus.gd" &amp;&amp; echo "SelectionManager script exists with input handling"</automated>
    - File `res://scripts/selection_manager.gd` exists with raycast logic.
    - Script emits `SignalBus.insect_selected` when clicking on a collider with collision mask 1.
    - Script emits `SignalBus.insect_deselected` when clicking empty space.
    - SelectionManager node exists in main scene.
  </verify>
  <done>SelectionManager handles mouse clicks and emits appropriate signals.</done>
</task>
